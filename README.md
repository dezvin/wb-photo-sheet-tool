# WB Photo Sheet

Простая статическая страница для сборки фотографий карточки Wildberries в листы по 9 изображений.

## Как использовать

Откройте `index.html`, вставьте артикул или ссылку на карточку WB и нажмите `Собрать`.

Страница ничего не сохраняет автоматически. Готовые JPG-листы скачиваются только по кнопке.

## Как выложить на GitHub Pages

1. Создайте пустой репозиторий на GitHub.
2. Загрузите в него файлы из этой папки:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
3. В настройках репозитория откройте `Pages`.
4. Включите публикацию из ветки `main`, папка `/root`.
5. Отправьте клиенту ссылку GitHub Pages.

## Ограничения

- Страница зависит от публичных серверов Wildberries.
- Если WB изменит адреса изображений или правила доступа из браузера, сборка может перестать работать.
- Листы собираются в памяти браузера; очень старые телефоны могут не справиться с карточками на 30+ фото.

## Ozon

Для Ozon есть отдельная страница с bookmarklet:

```text
ozon-bookmarklet.html
```

Она работает иначе: сначала пользователь сам открывает карточку Ozon, затем нажимает закладку. Код запускается внутри уже открытой страницы и собирает найденные фото в листы.
