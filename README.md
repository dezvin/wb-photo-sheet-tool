# Фото товара в листы 3×3

Статическая страница для подготовки фотографий карточки маркетплейса к проверке нейросетью.

## Что умеет

- Wildberries: принимает артикул или ссылку, скачивает крупные фото карточки и собирает JPG-листы по 9 фото.
- Ozon: даёт кнопку-закладку, которую пользователь запускает внутри уже открытой карточки товара.
- Готовые листы создаются в браузере. Сайт не хранит файлы и не отправляет их на сервер.

## Как использовать

Откройте `index.html` или опубликованную страницу GitHub Pages.

Для Wildberries:

1. Выберите `Wildberries`.
2. Вставьте артикул или ссылку.
3. Нажмите `Собрать листы`.
4. Скачайте готовые JPG-листы.

Для Ozon:

1. Выберите `Ozon`.
2. Перетащите кнопку `Ozon фото в листы` на панель закладок.
3. Откройте карточку Ozon.
4. Нажмите закладку на странице товара.
5. Проверьте найденные фото и соберите выбранные в листы.

## Публикация на GitHub Pages

Загрузите в репозиторий все файлы из этой папки и включите Pages из ветки `main`, папка `/root`.

Основная страница: `index.html`.

Отдельная страница `ozon-bookmarklet.html` оставлена как прямой адрес для установки Ozon-кнопки, но основной сценарий теперь находится на главной странице.

## Ограничения

- Wildberries-сценарий зависит от публичных адресов изображений WB.
- Ozon может менять страницу товара, поэтому перед сборкой пользователь видит найденные фото и может снять лишние галочки.
- Очень старые телефоны могут медленно собирать товары с большим количеством фото.
